Automotive Portfolio Website

This project is a personal portfolio website created using HTML, CSS, and JavaScript.

The website showcases professional experience, skills, and contact information in an interactive single-page layout inspired by the automotive industry.

 Features

- Interactive dashboard-style navigation
- Automotive themed design
- Skills section with progress bars
- Professional experience timeline
- Contact information section
- Responsive design

Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- GitHub Pages (for hosting)
